# NailingUpProject
노드 JS와 EJS 기반으로 프로젝트를 괸리하는  웹 서비스

#개요
 노드를 이용하여, 웹서비스를 구현해 본다. 
 기획서에 제출했던 본인이 만들어보고 싶은 웹싸이트를 구축해 본
다.

#사용기술
 프론트엔드 : HTML/CSS/JS/EJS/Bootstrap
 벡엔드 : Nodejs + MySQL

#기능
프로세스의 시각화, 친절한 인터페이스
색상의 통일성 , 레이아웃 분리
효율적인 프로젝트 관리를 위한 멤버 추가/삭제/수정 기능 추가
